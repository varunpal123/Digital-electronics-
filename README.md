# Arduino Uno (ATmega328P) Verilog Core 

Welcome to the **Arduino Uno Verilog Implementation** repository! This project provides an open-source, synthesizable Verilog Hardware Description Language (HDL) model of the Arduino Uno, focusing on the core ATmega328P 8-bit AVR microcontroller architecture. 

This repository is ideal for students, hobbyists, and digital design engineers looking to understand computer architecture, port Arduino functionality to FPGAs, or simulate AVR machine code at the Register Transfer Level (RTL).

---

## 🚀 Project Overview

The goal of this project is to recreate the internal behavior of the Arduino Uno board. Rather than using the physical silicon chip, this project allows you to synthesize the microcontroller on a Field Programmable Gate Array (FPGA) or run cycle-accurate simulations using standard EDA tools. 

It executes standard **AVR hex files** generated by the Arduino IDE, allowing you to run your `.ino` sketches directly on custom hardware.

---

## ✨ Key Features

* **Core Architecture:** Implementation of the 8-bit AVR RISC architecture.
* **Instruction Set:** Support for the standard ATmega instruction set (ALU operations, branching, memory access).
* **Memory Architecture:** * Instruction memory (Flash ROM equivalent).
    * Data memory (SRAM equivalent).
    * 32 x 8-bit general-purpose working registers.
* **Peripherals:**
    * ✅ **General Purpose I/O** (PORTB, PORTC, PORTD)
    * ✅ **Hardware UART** (for Serial communication)
    * ⏳ **8-bit & 16-bit Timers/Counters** with PWM (In Progress)
    * ⏳ **SPI and I2C (Wire)** interfaces (Planned)
    * ⏳ **Interrupt Controller** (Planned)
